## Prompt
```
{: .prompt-info}
```
where `info` can be replaced by `tip`, `warning` and `danger`.
> this is a tip
{: .prompt-tip}
> this is a info
{: .prompt-info}
> this is a warning
{: .prompt-warning}
> this is a danger
{: .prompt-danger}
